### 创建一个自己的centos

```bash
# 1.配置文件

FROM centos
MAINTAINER yanzx<209243446@qq.com

ENV MYPATH /usr/local
WORKDIR $MYPATH

RUN apt-get install vim
RUN apt-get install net-tools

EXPOSE 80


CMD echo $MYPATH
CMD echo "----end----"
CMD /bin/bash
# 2
docker  build -f mydockerfile-centos -t mycentos:0.1 .

# 3.成功

```
